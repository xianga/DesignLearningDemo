#### 存储过程和函数的区别
- 存储过程
    - 存储过程不允许执行return，但是通过out参数返回多个值
    - 存储过程的限制相对就比较少
- 函数
    - 函数只能通过return语句返回单个值或者表对象
    - 可以嵌入sql使用
    - 函数限制比较多，比如不能用临时表，只能用表变量
    
    
#### 什么情况下不走索引
- 使用不等于操作符 (<> !=)
- NULL值，where子句中出现 is null 或者is not null
- 当使用通配符作为查询字符串的第一个字符时
- 对于count(*)当索引字段有not null约束时走索引，否则不走索引
- where条件等号两边字段类型不同，不走索引
- 使用函数（索引不是基于函数的索引）
